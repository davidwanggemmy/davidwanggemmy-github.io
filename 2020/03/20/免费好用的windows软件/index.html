免费好用的windows软件推荐

listary

Listary是一个革命性的Windows搜索工具 这使得查找您的文件和启动应用程序的速度非常快，无论是对普通用户还是付费用户都是如此！!

只需点击几下，您就可以快速轻松地完成：

  • 从任何窗口或文件夹中搜索整个磁盘驱动器

  • 只需几个短按键即可快速找到所需的任何文件

  • 使用快速键盘命令即时查找和启动程序

  • 使用简单的上下文菜单命令从任何文件夹复制或移动文件到任何文件夹

  • 从任何地方打开包含特定程序的文件

  • 使用简单的键命令即时从文件管理器跳转到目标文件夹

  • 创建常用文件和应用程序的收藏夹列表以便快速访问

  • 右键单击任何搜索结果，以获得强大的Action + Context菜单

  • 快速浏览最近的应用和文件历史记录，继续在您上次停留的地方继续工作

  • 使用Listary的关键字查询引擎搜索网页

    下载方式：

    百度就有，很好用。

    附：在ubuntu上我试过albert（github）基本有一样的效果。可以减少你查找文件的时间，避免你找不到重要的文件。

VLC

简单，快速，强大

  • 播放所有内容 -文件，光盘，网络摄像头，设备和流。
  • 播放大多数编解码器而无需编解码器包 -MPEG-2，MPEG-4，H.264，MKV，WebM，WMV，MP3 …
  • 在所有平台上运行 -Windows，Linux，Mac OS X，Unix，iOS，Android …
  • 完全免费 -没有间谍软件，没有广告，也没有用户跟踪。

下载方式：

百度就有，很好用。

使用感受：比起总弹广告的播放器，这个功能强大而且干净舒服。

天若OCR and 白描 （文字识别）

免费的图片文字识别工具

天若ocr支持竖排识别，文字识别，公式识别，自带截图功能，网页文本内容不让拷贝的时候，除了用针对性的google插件就用文字识别吧。

白描也是类似的一款软件，手机版也非常好用

下载地址：https://github.com/AnyListen/tianruoocr/releases

字由：字体管理软件

这款软件不仅可以管理我们电脑中已经安装的字体，还可以安装免费的好看字体。在编辑文档的时候，点一下这个软件里你喜欢样式的字体，就可以将它添加到你的字体库里，以后方便使用。比以前下载好字体挪到指定文件夹的方法方便多了。

下载方法：百度

bandicam：好用的录屏软件

Bandicam(高清录制视频软件)是一款由韩国开发的高清游戏及视频录制的电脑工具，Bandicam，游戏录制工具、视频录制工具、高清视频录制专家，号称世界三大视频录制神器之一，它录制的视频文件不仅体积小，而且画质相当清晰，支持H.264预制高清，以较高的压缩率可录制超过分辨率高达2560×1600高画质视频（1080p全高清视频），录制的同时还能添加水印到视频中，录制后支持编辑视频，支持bmp、png 、jpeg等格式截图

破解版和绿色版非常好用，是录屏非常好的选择。
